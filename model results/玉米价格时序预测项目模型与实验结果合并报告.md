# 玉米价格时序预测项目模型与实验结果合并报告

## 第一部分：玉米价格项目模型与结果整理

玉米价格项目模型与结果整理

覆盖方向：价格回归、涨跌趋势分类、数据增强、传统模型与深度模型 | 整理日期：2026-06-30

### 1. 数据与任务口径

- 原始数据文件：C:\时序玉米\玉米价格原始数据.csv。时间范围 2016-06-29 到 2026-06-26，共 2426 行。

- 核心目标列：dce_corn_close。项目保留了开高低收、持仓量、成交量、基差、价差、天气、节假日等特征。

- 两类任务口径：价格回归，直接预测次日收盘价；趋势分类，预测次日相对当日是否上涨。

- 本报告只写已经落盘的结果文件和你在对话中确认过的实验，不额外补编不存在的数值。

### 2. 主要模型

| 模型 | 类型 | 一句话说明 |
| --- | --- | --- |
| Random Forest | 树模型回归 | 多棵树做集成，适合做传统基线和特征重要性分析。 |
| Diffusion-TS | 扩散式数据增强 | 先生成合成时序样本，再拿增强后的数据训练回归器。 |
| TimeAutoDiff | VAE + 潜空间扩散 | 比 Diffusion-TS 更强的数据增强方案，生成的合成样本质量更好。 |
| LSTM | 序列神经网络 | 用时间窗口学习短期动态，是本项目的主力深度回归模型。 |
| KAN | 可解释神经网络 | 用基函数式层做非线性拟合，作为新型深度基线。 |
| MLP / CNN | 基础神经网络 | 分别代表纯全连接和一维卷积时序基线。 |
| ARIMA | 经典统计模型 | 单变量自回归积分滑动平均模型，作为最朴素的价格回归对照。 |
| iTransformer | Transformer 变体 | 把时间序列转换成适合注意力学习的形式。 |
| XGBoost / LightGBM | 分类模型 | 用于涨跌方向预测，通常比深度模型更稳。 |
| Chronos | 开源权重模型 | 预训练时序模型，做零样本或轻量微调。 |
