# 玉米价格趋势预测项目

本项目面向可口可乐糖浆供应商的玉米采购场景。由于玉米是生产果葡糖浆的重要原材料，玉米价格波动会直接影响企业采购成本和生产决策。因此，本项目希望通过时序预测判断玉米价格未来一段时间内趋势，为供应商制定采购计划提供参考。

本项目以大连商品交易所玉米期货收盘价 `dce_corn_close` 为目标变量（增加特征版的目标变量是 target_up_30d_threshold），不直接预测未来具体价格，而是将任务转化为趋势分类问题：判断未来 30 个交易日内玉米价格涨幅是否超过 0.5%（阈值）。若未来 30 日收益率大于 0.5%，则记为上涨；否则记为未明显上涨。

项目主要使用玉米期货价格、成交量、持仓量、现货价格、基差、外盘价格、成本端变量和季节性变量等信息，构造收益率、移动均线、波动率、基差偏离、趋势持续性等特征。模型方面，计划使用 Logistic Regression、Random Forest、XGBoost、LightGBM、CatBoost、LSTM 等方法进行对比实验。

模型评价重点关注 Accuracy、Precision、Recall、F1-score、AUC 和混淆矩阵。其中，F1-score 和 AUC 作为主要评价指标，用于衡量模型对未来玉米价格上涨趋势的识别能力。最终，模型输出的上涨概率将转化为采购建议，帮助供应商判断是否需要提前采购、锁价或维持正常采购节奏。
