# 玉米价格趋势预测项目

本项目面向果葡糖浆供应链中的玉米采购场景，基于玉米期货、现货、基差、外盘、成本端和季节性变量等数据，预测未来玉米价格上涨趋势，为采购节奏、提前备货和锁价决策提供参考。

## 项目目标

- 将玉米价格预测转化为趋势判断问题
- 判断未来一段时间内玉米价格是否存在明显上涨可能
- 输出上涨概率，并辅助形成采购建议

## 预测任务

本项目不直接预测未来具体价格，而是关注价格趋势。模型根据历史价格、成交量、持仓量、现货价格、基差、外盘价格、成本端变量和季节性变量等特征，判断未来玉米价格是否上涨。

## 当前实验结果

| 方案 | Accuracy | Balanced Accuracy | AUC |
|---|---:|---:|---:|
| 双头 LSTM + 概率校准 | 0.915 | 0.940 | 0.927 |

## 仓库结构

- `Experimental Procedure/`：实验流程说明
- `Experimental Standards/`：实验标准口径

## 项目价值

该项目可以帮助供应商在玉米价格可能上涨时提前识别风险，为采购计划、库存安排和锁价决策提供数据支持。以月度玉米需求 10 万吨、30%-50% 弹性采购或锁价比例测算，模型预计每年可优化采购成本约 300 万至 500 万元；在价格冲击年份，避险价值可达到千万元级。
